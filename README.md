# meutudo - Mobile Developer

## Sobre

- **Descrição:** Torne-se um desenvolvedor mobile full stack! Domine a stack fundamental do bom desenvolvedor mobile de ponta a ponta com Node.js, React Native e JavaScript/TypeScript.
- **Formato:** Conteúdo estruturado com 10 projetos práticos e deploy real em lojas oficiais.
- **Destaques:** inclui um simulador de Mario Kart, um clone do ChatGPT e uma API da Champions League.

## Stack

- **Backend:** Node.js
- **Mobile:** React Native
- **Linguagens:** JavaScript / TypeScript

## Projetos do Bootcamp (exemplos)

- **Simulador de Mario Kart:** projeto prático com lógica de jogo e UI mobile.
- **Clone do ChatGPT:** exemplo de integração com APIs de IA e interface conversacional.
- **API da Champions League:** backend para gestão de partidas, classificação e estatísticas.
- **Total:** 10 projetos práticos ao longo do curso para compor seu portfólio.

## Como usar este repositório (rápido)

- Abra este diretório em seu terminal PowerShell (`pwsh.exe`).

- Rodar o exemplo Hello World:

```powershell
pwsh.exe
node 01-helloworld/hello.js
```

- Rodar o projeto exemplo no `02-meu-primeiro-projeto` (se houver `package.json`):

```powershell
pwsh.exe
cd 02-meu-primeiro-projeto
npm install
npm start
```
